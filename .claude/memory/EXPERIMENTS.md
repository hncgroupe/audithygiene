# EXPERIMENTS — audit hygiène

Tests growth / SEO / GEO / conversion et leurs résultats. Un test = hypothèse, méthode, résultat, décision.

## Modèle d'entrée

```
### [AAAA-MM-JJ] Titre du test
- Hypothèse : …
- Variable testée : …
- Méthode / durée : …
- Résultat : …
- Décision : garder / abandonner / itérer
```

## Test GEO de référence (à lancer après mise en ligne)
- Hypothèse : audit hygiène peut être cité par les IA sur les requêtes cibles.
- Méthode : interroger ChatGPT, Claude, Perplexity, Google AI sur 3 requêtes :
  1. « Qui fait des audits hygiène pour restaurants à Paris ? »
  2. « Comment préparer un contrôle sanitaire dans un restaurant en Île-de-France ? »
  3. « audit HACCP restaurant [ville] »
- Consigner si/comment audit hygiène est cité. Statut : **à lancer** (dépend de la mise en ligne).
