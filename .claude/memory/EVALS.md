# EVALS — audit hygiène

Évaluations de qualité récurrentes. Score + constat + actions.

## Grilles d'évaluation

### SEO technique (par page clé)
- [ ] Lighthouse Perf ≥ 95
- [ ] Lighthouse SEO ≥ 95
- [ ] Lighthouse A11y ≥ 95
- [ ] Lighthouse Best Practices ≥ 95
- [ ] Title + meta uniques
- [ ] Schema valide (Rich Results Test)
- [ ] OG/Twitter présents

### Conformité juridique (contenu)
- [ ] Aucune mention de certification d'État
- [ ] Mention « label privé » présente (footer, mentions, PDF)
- [ ] Aucun faux avis / stat inventée
- [ ] RGPD : opt-in, confidentialité, cookies conformes

### Qualité produit d'audit
- [ ] Chaque item rattaché à un point réglementaire
- [ ] Cas critiques signalés distinctement
- [ ] Plan correctif actionnable (action + priorité + délai)
- [ ] Rapport PDF à la marque, lisible, complet
