# Skill — competitor-analysis

## But
Comprendre les acteurs de l'audit hygiène / conseil HACCP en IDF pour se différencier.

## Quand l'utiliser
- Positionnement, pricing, contenu, SEO concurrentiel.

## Méthode
1. **Recenser** : cabinets d'audit/conseil HACCP, sociétés de mise aux normes, indépendants, plateformes.
2. **Analyser** : offres, prix affichés, positionnement, SEO (mots-clés, pages), preuves de confiance.
3. **Gaps** : ce qu'ils ne font pas bien (rapport, transparence, local, GEO).
4. **Différenciation** : outil propriétaire, rapport premium, label, SEO/GEO local.
5. **Documenter** dans `memory/research/concurrents.md`.

## Garde-fou
- Données observées réelles, sourcées. Pas de dénigrement diffamatoire.
- Ne pas copier de contenu protégé.
