# Skill — conversion-optimization

## But
Maximiser le taux visiteur → lead → RDV sur le site vitrine.

## Quand l'utiliser
- Concevoir/optimiser hero, formules, formulaire RDV, CTA.

## Méthode
1. **Hero** : promesse claire + CTA réservation visible immédiatement (above the fold, mobile compris).
2. **Qualification 2-3 questions** : type d'établissement, taille/couverts, urgent vs préventif → oriente vers la bonne formule.
3. **Formules lisibles** : valeur, contenu, durée, prix (ou « sur devis » si `TODO`). Réduire le choix paralysant.
4. **Formulaire RDV minimal** : champs nécessaires seulement, validation claire, consentement RGPD.
5. **Réassurance** : déroulé de l'audit, exemple de rapport, FAQ, cadre label privé.
6. **Friction** : mobile-first, vitesse, CTA répété, pas d'étape inutile.
7. **Mesure** : événements analytics (skill `analytics-strategy`), itérer (skill `ab-test` côté growth).

## Garde-fou
- Pas de dark patterns. Pas de fausse urgence inventée. Consentement RGPD réel.
