# Skill — seo-strategy

## But
Piloter le SEO technique et on-page pour viser le top sur « audit hygiène restaurant » en IDF.

## Quand l'utiliser
- Créer/auditer une page côté référencement.
- Définir titles, métas, structure Hn, schema, maillage.

## Méthode
1. **Technique** : sitemap.xml, robots.txt, canonical, OG/Twitter, images WebP/AVIF + alt, fonts optimisées, lazy-load. Viser Lighthouse ≥ 95.
2. **On-page** : 1 intention par page, title unique (≤ ~60 car.), meta description (≤ ~155 car.), un seul H1, hiérarchie Hn propre, réponse directe en intro.
3. **Schema.org** : `LocalBusiness`, `Service`, `FAQPage`, `BreadcrumbList` (et `AggregateRating` seulement si vrais avis).
4. **Maillage interne** : silos (national → département → ville), ancres descriptives.
5. **Contenu** : profondeur réelle, E-E-A-T (skill `eeat-trust-signals`), zéro duplication.

## Garde-fou
- Pas de promesse de rang. Honnêteté sur le délai SEO.
- Pas de bourrage de mots-clés. Pas de contenu dupliqué entre pages locales.
- Respect du cadre label privé dans les métas et titres.
