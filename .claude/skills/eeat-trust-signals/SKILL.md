# Skill — eeat-trust-signals

## But
Maximiser les signaux E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness) honnêtes.

## Quand l'utiliser
- Rédiger pages méthode, à-propos, blog, fiches auteur.

## Méthode
1. **Expérience** : montrer la pratique réelle (déroulé d'un audit, exemple de rapport anonymisé).
2. **Expertise** : contenu réglementaire précis et juste (HACCP, PMS, Paquet hygiène), auteur identifié avec qualif réelle.
3. **Autorité** : page méthode détaillée, glossaire, mentions presse réelles (jamais inventées).
4. **Confiance** : transparence sur le cadre label privé, mentions légales, politique de confidentialité, coordonnées, processus clair.
5. **Preuves** : vrais avis (si collectés), vraies études de cas (avec accord).

## Garde-fou
- Tout signal doit être réel (rules `no-fake-content`, `label-prive-cadre-juridique`).
- Pas de fausse qualification, pas de faux logo, pas de faux avis.
