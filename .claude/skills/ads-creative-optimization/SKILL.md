# Skill — ads-creative-optimization

## But
Concevoir des créas et annonces (Search/Display) performantes pour l'acquisition payante.

## Quand l'utiliser
- Préparer des annonces Google Ads / Meta pour l'audit hygiène.

## Méthode
1. **Search** : titres alignés à l'intention (« audit hygiène restaurant [ville] »), USP claire, CTA, extensions (sitelinks, accroches, lieu).
2. **Angles** : anticiper le contrôle, sérénité, rapport clair + plan d'action, label de confiance.
3. **Cohérence** annonce ↔ landing page (message match).
4. **Variantes** à tester (RSA : plusieurs titres/descriptions).
5. **Conformité** : cadre label privé respecté dans chaque annonce.

## Garde-fou
- ⏸️ Toute **activation de budget = validation humaine** (rule `validation-humaine`).
- Pas de promesse de garantie officielle. Pas de stat inventée.
