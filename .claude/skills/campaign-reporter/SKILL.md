# Skill — campaign-reporter

## But
Produire des comptes-rendus clairs et actionnables (SEO, ads, email, GEO).

## Quand l'utiliser
- Faire le point périodique sur une campagne/canal.

## Méthode
1. **Structure** : objectif → résultat chiffré → écart vs cible → cause → action recommandée.
2. **Données réelles** seulement (Search Console, GA4, Brevo, DB).
3. **Honnêteté** : signaler les contre-performances, pas seulement les wins.
4. **Synthèse exécutable** : 3 décisions max par rapport.
5. **Archiver** dans `docs/channels/` et `memory/EXPERIMENTS.md`.

## Garde-fou
- Pas de chiffre inventé (rule `no-fake-content`). Si donnée manquante, le dire.
